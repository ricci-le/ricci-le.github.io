The knowledge graph is a visual representation of how notes connect to each other.

## How It Works

Each note is a node, and connections are edges. When you link to another note using [[wikilink]] syntax, a connection is created.

## Features

- Zoom in and out
- Drag nodes to rearrange
- Click nodes to view notes
- Color-coded by tags
