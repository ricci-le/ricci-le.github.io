Tags help organize and categorize notes.

## How to Use Tags

Add tags to the frontmatter of your markdown files:

```yaml
---
tags: [react, typescript, frontend]
---
```

## Popular Tags

- `react` - React related notes
- `typescript` - TypeScript notes
- `backend` - Backend development
- `devops` - DevOps and deployment
